# once_javiera

Para comenzar es necesario abrir una cuenta en el GitHub. 

**Conocer mis credenciales en github**

Es importante saber el nombre con el cual registre mi cuenta al igual de mi contraseña. 

**Conocer mi URL de github**

Este es el link que tiene tu cuenta. 
Como por ejemplo este es el mío: 
https://github.com/toniamlg.

**Cómo crear un repositorio**

Ingreso a mi cuenta, en la parte superior se encuentra: Repositories, el lugar donde se guardan los trabajos. Al estar en esta parte escojo la opción “New” así creando un repositorio, lo puedo llamar de la forma que yo quiera así encontrando en la parte de abajo la opción de confirmación para crearlo. Tenemos la opción de crearlo oficialmente y nos lleva directamente a un Visual Studio Code.
